# Jogo de Contas – Desafio de Matemática

Este projeto é um pequeno jogo de perguntas matemáticas em que o jogador precisa resolver contas de adição. A cada partida, um número aleatório de desafios é gerado, tornando o jogo mais divertido e imprevisível.

## Descrição

O programa sorteia entre 1 e 10 perguntas, cada uma com uma soma aleatória entre 1 e 10.
O jogador deve responder corretamente para avançar — respostas erradas fazem a pergunta se repetir até acertar.

Ao final, o jogador recebe uma mensagem personalizada de vitória.
